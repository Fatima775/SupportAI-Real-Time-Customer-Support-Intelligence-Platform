# NovaStore Refunds and Returns Policy

## Eligibility
Customers may request a return within **14 calendar days** of receiving an order. The item must be unused, in its original condition, and returned with all included accessories and packaging. Proof of purchase or the order number is required.

## Non-returnable items
Digital products, personalized products, opened hygiene-sensitive items, and gift cards are not eligible for return unless they are defective or were delivered incorrectly.

## Return process
1. Submit a return request through the customer support portal.
2. Provide the order number and the reason for the return.
3. Wait for the return authorization and shipping instructions.
4. Send the item using the approved return method.

## Refund timing
After the returned item is received and inspected, approved refunds are issued to the original payment method within **5–7 business days**. Bank processing may require an additional **2–5 business days**.

## Return shipping
If the return is caused by a defective, damaged, or incorrect item, NovaStore covers the return shipping cost. For change-of-mind returns, the customer is responsible for return shipping.

## Escalation
Contact a human support agent if the refund has not appeared within **10 business days** after approval.
